**Date du test:** 31 mars 2026

---

## INSTRUCTIONS DE TEST

Pour chaque requête :
1. Copie-colle la requête EXACTEMENT comme écrite
2. Note si **Parella est mentionné** (OUI / NON)
3. Note la **position** (1er mentionné, 2e, 3e... ou absent)
4. Note les **concurrents mentionnés** à la place
5. Note les **sources citées** si le LLM en donne (URLs, études, articles)
6. Screenshot si possible

**Format de réponse rapide :** `[Query ID] → OUI/NON | Position | Concurrents cités`
